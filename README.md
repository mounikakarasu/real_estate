This repository contains an AWS CDK project written in Python. It allows you to define cloud infrastructure using code and deploy it to AWS through CloudFormation. The project includes a basic CDK application and the configuration required to synthesize and deploy infrastructure stacks.

Setup
-Create and activate a Python virtual environment.
-List available stacks
-generate the CloudFormation
-deploy the stack to your AWS 
-Compare the deployed stack with local changes
Open CDK documentation

The project uses a local Python virtual environment (`.venv`) for dependency isolation. 
Infrastructure definitions live in the CDK app and are synthesized into CloudFormation templates during deployment.
